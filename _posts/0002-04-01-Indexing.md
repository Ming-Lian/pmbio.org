### Create reference index for use with BWA

Use bwa index to create an index for alignment

- Runtime: 70min


```bash
cd ~/workspace/data/raw_data/references
bwa index ~/workspace/data/raw_data/references/GRCh38_full_analysis_set_plus_decoy_hla.fa
```

### Index files for use with GATK

```bash
#SNP calibration call sets - dbsnp, hapmap, omni, and 1000G
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file ~/workspace/data/raw_data/references/Homo_sapiens_assembly38.dbsnp138.vcf.gz
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file ~/workspace/data/raw_data/references/hapmap_3.3.hg38.vcf.gz
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file ~/workspace/data/raw_data/references/1000G_omni2.5.hg38.vcf.gz
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file ~/workspace/data/raw_data/references/1000G_phase1.snps.high_confidence.hg38.vcf.gz

#Indel calibration call sets - dbsnp, Mills
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file /home/ubuntu/data/reference/Homo_sapiens_assembly38.known_indels.vcf.gz
gatk --java-options '-Xmx24g' IndexFeatureFile --feature-file /home/ubuntu/data/reference/Mills_and_1000G_gold_standard.indels.hg38.vcf.gz

```
